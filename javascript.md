## JavaScript

### Hello World:
```javascript
console.log('Hello World');
```

### Variables and constants:
```javascript
var foo = 'bar';
let foo = 'bar';
const foo = 'bar';
```

### Data types:
```javascript
const name = 'John';
let age = 30;
const rating = 7.5;
const isBlue = true;

const fruits = ['banana', 'apple', 'watermelon'];
```
